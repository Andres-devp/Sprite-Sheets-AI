## 2. LÓGICA DE NEGOCIO Y BACKEND (CRÍTICO)

La aplicación no es solo una interfaz estática; debe ser funcional. Implementa las siguientes mecánicas:

### A. AI Generate (Hugging Face Img2Img)
- El formulario del Paso 1 permite al usuario subir una "Imagen de Referencia" y escribir un prompt de texto.
- Al presionar "Generate", el cliente debe convertir esa imagen a Base64 y enviarla junto con el texto a un endpoint interno: `/api/generate`.
- **El Backend (`route.ts`)**: Recibe el Base64 y el prompt. Concatena el prompt con un system prompt oculto ("Estilo pixel art, cuadrícula, fondo verde croma #00FF00 sólido"). 
- El servidor hace un `POST` seguro a la **Hugging Face Serverless Inference API** usando un modelo de `image-to-image` (Img2Img). Debe incluir el header de autorización leyendo `process.env.HF_TOKEN`. El resultado generado se devuelve al cliente para mostrarse en el Paso 2.

### B. Remove Background (Chroma Key Local)
- **NO uses APIs externas para esto.** - Toda la lógica debe ocurrir en el cliente usando HTML5 `<canvas>`.
- Al abrir una imagen en el editor o en el panel de "Background Removal", dibuja la imagen en un canvas oculto, obtén el `ImageData`, y recorre los píxeles.
- El slider de "Tolerance" debe calcular la diferencia matemática entre el color del píxel actual y el color objetivo (verde brillante `#00FF00` o negro, según se detecte o seleccione). Si la diferencia es menor a la tolerancia, cambia el canal Alpha de ese píxel a `0` (transparente) y redibuja la imagen.

### C. Video to Sprite (Extracción)
- Cuando el usuario sube un archivo de video local, reprodúcelo de forma oculta en el navegador.
- Usa `requestAnimationFrame` o un bucle temporizado para capturar fotogramas dibujando el `<video>` en un `<canvas>` en intervalos específicos según el slider de "Frame Rate" (FPS).
- **Exportar**: El botón de "Export" debe permitir compilar los fotogramas seleccionados en un archivo `.zip` usando la librería `jszip`, o generar una sola imagen PNG que acomode todos los fotogramas en una cuadrícula basándose en el slider de "Columns".
